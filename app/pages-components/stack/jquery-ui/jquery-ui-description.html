jQuery UI

Interactions · Widgets · Effects · Themes

jQuery UI es una colección de interacciones, widgets, efectos y temas construida sobre jQuery. Proporciona componentes de interfaz de usuario listos para producción como Datepicker, Dialog, Accordion, Tabs, Autocomplete y muchos más.

Su sistema de temas mediante ThemeRoller permite personalizar la apariencia visual de todos los widgets con un único tema CSS coherente. jQuery UI se divide en cuatro categorías: Interactions (arrastrar, soltar, redimensionar), Widgets (componentes UI), Effects (animaciones avanzadas) y Utilities (herramientas de posicionamiento).

Categorías Principales

🖱️

Interactions

Añaden comportamientos de interacción a cualquier elemento: Draggable (arrastrar), Droppable (zonas de suelta), Resizable (redimensionar), Selectable y Sortable (reordenar elementos con drag & drop).

🧩

Widgets

Componentes UI accesibles listos para usar: Accordion, Autocomplete, Button, Datepicker, Dialog, Menu, Progressbar, Slider, Spinner, Tabs y Tooltip.

✨

Effects

Extiende las animaciones de jQuery con efectos adicionales: blind, bounce, clip, drop, explode, fold, highlight, puff, shake, slide y transfer. También añade easing avanzado.

🎨

ThemeRoller

El sistema de temas de jQuery UI aplica un CSS unificado a todos los widgets mediante clases como .ui-widget, .ui-state-hover y .ui-corner-all. ThemeRoller permite crear y descargar temas personalizados en línea.

♿

Accesibilidad (WAI-ARIA)

Los widgets de jQuery UI implementan los patrones WAI-ARIA de accesibilidad, añadiendo automáticamente atributos role, aria-* y soporte completo de navegación por teclado a todos sus componentes.

🔧

Widget Factory

$.widget() es la base de todos los widgets de jQuery UI. Permite crear plugins con estado, opciones configurables, métodos públicos y privados, herencia y un ciclo de vida consistente (create, destroy, enable, disable, option).
