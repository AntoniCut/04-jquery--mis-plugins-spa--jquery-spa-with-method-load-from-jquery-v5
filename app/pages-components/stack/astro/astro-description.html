Astro

El framework web que prioriza el contenido

Astro es un framework web moderno diseñado para construir sitios web rápidos centrados en el contenido. Utiliza una arquitectura Islands Architecture que permite enviar JavaScript solo cuando es necesario.

Astro soporta múltiples frameworks de UI (React, Vue, Svelte, SolidJS) en un mismo proyecto, compilando a HTML estático al momento de build. Ideal para blogs, documentación y landing pages de alto rendimiento.

Características Principales

🏝️

Islands Architecture

Hidrata solo los componentes interactivos, manteniendo el resto estático. JavaScript se carga solo cuando es necesario, optimizando el rendimiento.

⚡

Zero JavaScript

Por defecto, Astro no envía nada de JavaScript al navegador. Los componentes interactivos se hidratan selectivamente según sea necesario.

🎯

Multi-Framework

Usa React, Vue, Svelte, SolidJS o vanilla JavaScript en el mismo proyecto. Elige la mejor herramienta para cada componente sin restricciones.

📝

Content Collections

Organiza y valida tu contenido con type-safe schemas. Ideal para blogs, documentación y sitios basados en contenido Markdown o MDX.

🔌

Integraciones

Tailwind, MDX, image optimization, sitemap generation, y decenas de integraciones oficiales que extienden Astro fácilmente.

🧪

Server Side Rendering

Alterna entre estático (SSG), servidor (SSR) o híbrido según necesites. Despliega en cualquier hosting: Vercel, Netlify, Cloudflare, y más.
