jQuery

Write less, do more

jQuery es una librería JavaScript rápida, ligera y repleta de funciones. Simplifica el recorrido y manipulación del DOM, el manejo de eventos, las animaciones y las peticiones AJAX con una API que funciona en todos los navegadores.

Lanzada en 2006 por John Resig, jQuery fue durante años la librería JavaScript más utilizada del mundo. Aunque el JavaScript moderno (ES6+) ha reducido su necesidad, jQuery sigue siendo clave para entender el ecosistema web, para trabajar con proyectos legacy y como base de frameworks como jQuery UI y Bootstrap 3.

Características Principales

🔍

Selección del DOM

El selector $() usa la misma sintaxis que los selectores CSS para encontrar elementos: $('#id'), $('.clase'), $('div > p'). Devuelve un objeto jQuery con todos los métodos disponibles.

✏️

Manipulación del DOM

Métodos como .html(), .text(), .val(), .append(), .prepend(), .remove() y .addClass() permiten modificar el contenido y los atributos del DOM de forma concisa.

🖱️

Eventos

.on(), .off() y .trigger() ofrecen un sistema de eventos unificado. El patrón event delegation con $(parent).on('click', '.hijo', fn) es especialmente potente para elementos dinámicos.

✨

Animaciones

Métodos como .fadeIn(), .fadeOut(), .slideUp(), .slideDown() y .animate() facilitan la creación de efectos visuales con control de duración, easing y callbacks al completarse.

🌐

AJAX

$.ajax(), $.get() y $.post() abstraen el XMLHttpRequest nativo, devolviendo un objeto deferred compatible con promesas para encadenar .then(), .done() y .fail().

🔗

Chaining

La mayoría de los métodos jQuery devuelven el mismo objeto jQuery, permitiendo encadenar llamadas: $('p').addClass('active').fadeIn(300).css('color', '#21759b'), lo que hace el código más legible y compacto.
