CSS3

El lenguaje del diseño web

CSS (Cascading Style Sheets) es el lenguaje que describe cómo se deben mostrar los elementos HTML en pantalla, papel u otros medios. Controla el color, la fuente, el espaciado, el diseño y prácticamente todo aspecto visual de una página web.

CSS3 introduce módulos potentes como Flexbox, Grid, animaciones, variables (Custom Properties), filtros y media queries, permitiendo crear interfaces modernas, responsivas y visualmente atractivas con menos código.

Características Principales

🎯

Selectores y Especificidad

CSS ofrece selectores de elemento, clase (.clase), ID (#id), atributo, pseudoclases (:hover, :nth-child) y pseudoelementos (::before, ::after).

📦

Box Model

Todo elemento en CSS es una caja compuesta por content, padding, border y margin. La propiedad box-sizing: border-box simplifica el cálculo de tamaños.

🔀

Flexbox

Flexbox permite alinear y distribuir elementos en una dimensión (fila o columna) con propiedades como justify-content, align-items y flex-wrap.

⚏

CSS Grid

Grid Layout ofrece un sistema de diseño bidimensional poderoso, con grid-template-columns, grid-template-rows y la unidad fr para layouts complejos de forma sencilla.

🎨

Variables CSS (Custom Properties)

Las variables CSS (--nombre: valor) permiten definir valores reutilizables en :root o ámbitos locales, facilitando el theming y el mantenimiento del código.

✨

Animaciones y Transiciones

Con transition y @keyframes + animation, CSS permite crear efectos visuales fluidos y dinámicos sin necesidad de JavaScript.
